## PHASE 2: Subject Extraction (ลบพื้นหลัง)

## PHASE 3: Golden Ratio Facial Mapping (วัดสัดส่วน)

### 3.1 ระบบวัด 3 ชั้น
1. **Rule of Fifths (แนวตั้ง 5 ส่วน):** ขอบหน้า→หางตา→หัวตา→หัวตา→หางตา→ขอบหน้า
2. **Rule of Thirds (แนวนอน 3 ส่วน):** ไรผม→คิ้ว→ฐานจมูก→คาง
3. **Golden Ratio (φ=1.618):** Face W:L, Nose:Mouth, Lip ratio, Eye:Face, Jaw:Cheek

### 3.2 ค่าที่ต้องวัดและบันทึก

| ค่า | วัดจาก | ค่าหมอกวาง | เทียบ φ |
|---|---|---|---|
| Face Width:Length | หน้าตรง | 0.70 | +13% กว้างกว่า golden |
| Nose:Mouth | หน้าตรง | 0.63 | ≈ φ (สมบูรณ์แบบ!) |
| Upper:Lower Lip | หน้าตรง | 0.59 | ≈ φ (ใกล้เคียง) |
| Eye:Face Width | หน้าตรง | 0.21 | = 1/5 rule (perfect) |
| Jaw:Cheekbone | หน้าตรง | 0.77 | กรามชัด |
| Face Thirds | หน้าตรง | 0.90:1.00:1.05 | หน้าผากสั้น |
| Head:Body | เต็มตัว | 7.3:1 | 160cm |

### 3.3 Profile Measurements
- **Forehead projection:** ~1.3-1.4x chin baseline
- **Nasolabial angle:** ~95°
- **Profile classification:** mild Class II convexity
- **Ricketts E-line:** upper lip ~3mm behind, lower ~2mm behind

## PHASE 6: Iterative Refinement (ปรับแต่งจุดเฉพาะ)

### 6.1 ปัญหาที่มักเจอ & วิธีแก้

| ปัญหา | สาเหตุ | Prompt Fix | วิธี |
|---|---|---|---|
| หน้าผากยุบ | AI ทำ profile แบน | "prominent convex forehead projection" | Regenerate |
| คอยาว | AI ยืดคอ | "SHORT neck, petite frame [height]cm" | Regenerate |
| คางแหลม | AI ทำ V-shape | "rounded soft chin, Jaw:Cheek=0.77" | Modify |
| หน้ากว้าง | AI ทำหน้ากลม | "oval face, Face W:L=0.70" | Modify |
| ปาก/ฟันผิด | ขาด ref ยิ้มดี | เพิ่ม REF_EXPRESSION ที่เห็นฟัน | Regenerate |
| หัวใหญ่เกิน | AI ขยายหัว | "head-to-body ratio 7.3:1" | Regenerate |
| หน้าไม่เหมือนเลย | ref น้อย | เพิ่ม ref ให้สูงสุด 12-14 รูป | Regenerate |

### 6.2 Modify vs Regenerate Decision
- **ปรับเล็กน้อย** (หน้าเรียว, คางกลม, สีผิว): ใช้ Modify Image → เร็ว รักษาส่วนอื่น
- **ปรับใหญ่** (หน้าไม่เหมือน, สัดส่วนผิดมาก): Generate ใหม่ → แม่นกว่า
- ลอง Modify ก่อนเสมอ ถ้าไม่ได้ค่อย Regenerate

## PHASE 7: Scene Placement (วางในฉาก)
- ใช้ Approved Reference Sheet + Scene Base Image
- Modify ภาพฉากโดยอ้าง approved face refs
- ระบุชุด, ท่าทาง, expression ชัดเจน
- QA biometric ในฉาก: 85%+

## PHASE 8: Video Production

### Pipeline:
1. **TTS** — สร้างเสียงพากย์จากสคริปต์ (เลือก voice ที่เหมาะ)
2. **Keyframes** — สร้างภาพ character ในแต่ละ segment
3. **Video Generation** — สร้างวิดีโอจาก keyframes
4. **Lipsync** — ซิงค์ปากกับเสียง
5. **Infographics** — สร้างภาพประกอบ/กราฟ
6. **Render** — รวม text overlays + subtitles + BGM + stitch

## KEY LEARNINGS

### ✅ สิ่งที่ได้ผล
1. 12+ references ใน Nano Banana Pro → หน้าเหมือนที่สุด
2. Golden Ratio วัดตัวเลข → QA แม่นกว่าดูด้วยตาเปล่า
3. Chain reference ทีละรูป → consistency สูง
4. Modify เฉพาะจุด → เร็วกว่า generate ใหม่ทั้งรูป
5. ข้อมูลจากเจ้าตัว (ส่วนสูงจริง) → สัดส่วนถูกต้อง
6. ศึกษารูปต้นฉบับละเอียดก่อน generate → ลดรอบแก้ไข
7. ทดสอบหลาย models → เลือกตัวที่เหมาะกับแต่ละงาน

### ❌ สิ่งที่ไม่ได้ผล
1. ยัดทุกอย่างใน prompt เดียว → AI สับสน
2. Reference น้อย (1-3 รูป) → หน้าไม่เหมือน
3. ไม่ระบุ negative constraints → AI แต่งสวย/genericize
4. ไม่มี profile จริง → หน้าผากผิดรูปทุกครั้ง
5. ประมาณส่วนสูงเอง → head:body ratio ผิด
